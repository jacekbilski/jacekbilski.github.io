Improving your software by Gernot Starke

Jacek Bilski
value
length: 51:04

It’s probably one of quite a few videos about that topic, but probably the only one publicly available that I know of. The topic is: value, as in how “business” perceives what we, engineers are doing. It’s probably the one thing that turns you from a mere code writer to a professional software engineer. At least this is how I feel now that I understand the idea. My recommendation to all developers, if you want to advance your career, learn about value, everything else are implementation details.
